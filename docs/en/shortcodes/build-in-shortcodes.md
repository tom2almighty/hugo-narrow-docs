---
title: Build in Shortcodes
---

Hugo has built in some shortcodes:

- Details
- Figure
- Gist
- Highlight
- Instagram
- Param
- QR
- Ref
- Relref
- Vimeo
- X
- YouTube

These shortcodes usually unstable and could be changed at anytime, Hugo Narrow lacks special optimizations for these shortcodes.

See [Hugo Documentation](https://gohugo.io/shortcodes/) learn how to use these build in shortcodes.
